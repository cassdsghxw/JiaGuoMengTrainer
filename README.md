## 需求
- windows
- 安卓模拟器
- [按键精灵手机助手](http://www.mobileanjian.com/) / [按键精灵](http://www.mobileanjian.com/)
- 分辨率 540 × 960 (暂未做适配, 也懒得做)

## 兼容
> 理论上移动设备只要按键精灵支持, 本脚本就支持, 但是会有分辨率的问题, 所以大家最好还是用模拟器挂机吧

- 模拟器 `兼容`
- Android手机 `未测试`
- iPhone `未测试`

## 格子

> 九宫格, 从左到右从上到下, 格子序号从0开始

| 6 | 7 | 8 |  
| 3 | 4 | 5 |  
| 0 | 1 | 2 |
